# Changelog

## [1.0.0] - 2025-03-10

### Added

- **idea-interviewer**: 蘇格拉底式提問工具,產出 Idea Brief + Persona
- **prd-builder**: 從 Idea Brief 展開為 User Stories 與 PRD
- **challenger**: 批判性審查工具,支援多視角 Review
- **flow-mapper**: 產生 Sitemap、User Flow、Page Inventory
- **prototyper**: 將資訊架構轉化為可互動原型
- **architect**: PM 視角的技術方向建議工具
- **component-naming-skill**: 基於字根字首規範的元件命名助手

### Infrastructure

- 建立 Claude Code Plugin Marketplace 結構
- 新增 marketplace.json 配置檔
